# Alura Store — Challenge (Sr. Juan)

En este desafío ayudo al **Sr. Juan** a decidir **qué tienda de Alura Store debe vender** para iniciar un nuevo emprendimiento.  
La decisión se basa en el análisis de datos de **ventas, reseñas y rendimiento** de **4 tiendas**.

## Objetivo
Identificar la **tienda menos eficiente** y justificar la recomendación con métricas y visualizaciones.

## Qué se analiza
- Ingresos por tienda
- Categorías más vendidas
- Productos más vendidos
- Reseñas / calificación promedio
- Envío promedio

## Gráficas
Se incluyen **mínimo 3 gráficos diferentes** (por ejemplo: barras, circular, dispersión, etc.) para comparar tiendas.

## Herramientas
- Python
- Pandas
- Matplotlib
- Google Colab
